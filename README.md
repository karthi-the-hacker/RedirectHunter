## Installation

Install RedirectHunter with npm

```bash
  npm install redirecthunter -g
```

## Usage

Example for single url

```bash
  RedirectHunter -u http://example.com
```

## Usage

Example for list of urls

```bash
  RedirectHunter -l urls.txt -o out.txt
```
